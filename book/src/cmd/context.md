# context

context相关命令。

```plaintext
$ cldi help context
cldi-context
Context commands

USAGE:
    cldi context <SUBCOMMAND>

OPTIONS:
    -h, --help    Print help information

SUBCOMMANDS:
    save
    list       list contexts
    delete
    default    set a context as default and switch current context to it
    help       Print this message or the help of the given subcommand(s)
```
